DESCRIÇÃO:

A disciplina Modelagem de Banco de Dados oferece uma formação completa e sistemática para profissionais que desejam dominar a arte e a ciência de estruturar dados de forma eficiente, íntegra e escalável. Esta disciplina foi desenvolvida para capacitar estudantes desde os conceitos fundamentais até técnicas avançadas de implementação, proporcionando as competências essenciais para projetar sistemas de informação robustos que atendam às demandas organizacionais contemporâneas.

TECNOLOGIAS: 

Para por em prática o conceito de modelagem de banco de dados utilizamos algumas plataformas para o desenvolvimento do projeto como por exemplo: brModelo para por em prática a parte do projeto conceitual e o PostgreSQL para a parte do projeto físico da cidade inteligente com o tema TRÂNSITO INTELIGENTE.

OBJETIVO:

Através do projeto de "Cidade Inteligente" escolhemos uma área da cidade, que no caso foi o trânsito, e desenvolver que ela possa se automatizada. Assim, por em prática os conceitos de Modelo Entidade-Relacionamento (MER) e Diagrama Entidade-Relacionamento (DER), utilizando ferramentas profissionais de modelagem, transformação para o lógico e em seguida a tranformação para o físico utilizando o PostgreSQL.
